- platform package: `@canghe_ai/wechat-cli-darwin-arm64@0.2.4`
- local install path: `/usr/local/lib/node_modules/@canghe_ai/wechat-cli`

- `bin/wechat-cli.js`

- `node_modules/@canghe_ai/wechat-cli-darwin-arm64/bin/wechat-cli`
